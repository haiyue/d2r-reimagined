## 格式约定

| 标记 | 含义 |
|------|------|
| `V` | 原版（Vanilla）物品 |
| `M` | MOD 新增物品 |
| `req N` / `lv N` | 需求等级 N |
| `code:(xxx)` / `(xxx)` | 物品内部代码 |
| `rN` | 稀有度级别 |
| `[N]` / `[X]` / `[E]` | 普通 / 扩展 / 精英 级别 |
| `afLv` | 词缀等级（可出现的物品等级范围） |
| `{N件}` / `{全}` | 套装 N 件触发 / 完整套装触发 |
