### 1.1 腐化宝珠 (Orb of Corruption)
**功能**：将装备腐化，添加随机属性词缀，同时标记为"已腐化"。

**输入**：任意装备 + 腐化宝珠

**可用部位**：护符、腰带、靴子、手套、头盔、护甲、盾牌、单手武器、双手武器

**几率分布**（以护符为例）：
| 几率 | 效果 |
|------|------|
| 100% | +1 所有技能 |
| 94% | +20% 强化伤害，+10% 提高攻击速度 |
| 89% | +5% 所有元素技能伤害，+10% 更快速施法 |
| 83% | -5% 敌人元素抗性，+10% 更快速施法 |
| 77% | +10 所有属性 |
| 71% | +10% 所有抗性，+2% 所有最大抗性 |
| 65% | 5% 格挡率提高，+15% 更快速格挡 |
| 59% | 物理伤害降低 5%，魔法抗性 +5% |
| 53% | +3 每次消灭获得法力，+50 法力 |
| 46% | +3 每次消灭获得生命，+50 生命 |
| 21% | 仅标记已腐化 |
| 11% | 降级为稀有 |
| 1% | 降级为魔法 |

> 不同装备部位的几率效果略有不同

### 1.2 灌注之球 (Orb of Infusion)
**功能**：将特定品质的装备升级为稀有装备。

**输入**：
| 来源品质 | 结果 |
|----------|------|
| 低品质（破损） + 灌注之球 | → 稀有 |
| 普通品质 + 灌注之球 | → 稀有 |
| 高品质（超强） + 灌注之球 | → 稀有 |
| 稀有魔法箭袋/弩箭袋 + 灌注之球 | → 稀有 |

**可用部位**：任意武器、任意护甲、箭袋/弩箭盒

### 1.3 转换之球 (Orb of Conversion)
**功能**：将稀有装备转换为同底材的暗金装备（如果有该底材的暗金版本）。

**输入**：稀有装备 + 转换之球
**可用部位**：护符、任意护甲、戒指、任意武器、箭袋、弩箭盒

### 1.4 集聚之球 (Orb of Assemblage)
**功能**：将稀有装备转换为同底材的套装装备（如果有该底材的套装版本）。

**输入**：稀有装备 + 集聚之球
**可用部位**：护符、任意护甲、戒指、任意武器

### 1.5 镶嵌之球 (Orb of Socketing)
**功能**：为无孔的任意武器/护甲打 1 个孔。

**输入**：任意武器/护甲 + 镶嵌之球

> 不能用于已有孔的装备

### 1.6 暗影之球 (Orb of Shadows)
**功能**：将非无形装备变为无形（无法修复），提高基础属性约 50%。

**输入**：非无形装备 + 暗影之球
**可用部位**：单手武器、双手武器、长矛/长柄、任意护甲

### 1.7 重生之球 (Orb of Renewal)
**功能**：将装备变为白色/普通品质（去除所有魔法属性），可用于底材制作。

## 四、制作系统 (Item Crafting)

> 将魔法/稀有/暗金装备 + 符文 + 宝石/珠宝 合成为指定属性的装备

### 4.1 魔法装备制作 (Magic Crafting)
**输入**：魔法装备 + 伊司符文(6#) + 艾尔符文(1#)
**效果**：出固定词缀的魔法装备

### 4.2 稀有装备制作 (Rare Crafting)
**输入**：稀有装备 + 灌注之球 + 拉尔符文(8#) + 宝石
**效果**：生成新属性稀有装备

### 4.3 高级制作 (Advanced Crafting)
**输入**：稀有装备 + 稀有珠宝 + 普尔符文(21#) + 艾尔符文(1#) + 对应颜色宝石
**效果**：根据宝石颜色获得不同属性：
- 红宝石 → 火焰系
- 蓝宝石 → 冰霜系
- 绿宝石 → 毒素系
- 黄宝石 → 闪电系
- 钻石 → 物理系
- 骷髅石 → 通用系

**超级制作**：暗金珠宝 + 伐克斯(26#) + 罗(28#) + 骷髅石 → 职业专属制作装备

## 六、附魔系统 (Item Enchantment)

### 6.1 常规附魔
**输入**：装备 + 对应颜色宝石
**消耗**：宝石袋中的宝石点数（每次 -2～-10 点）
**效果**：随机获得附魔属性，每件装备有最大附魔次数

附魔属性示例（不同宝石类型出不同属性）：
| 宝石类型 | 可能属性 |
|----------|----------|
| 黄宝石 | MF%、金币%、照亮范围 |
| 红宝石 | 生命、生命恢复、火焰抗性 |
| 蓝宝石 | 法力、法力恢复、冰霜抗性 |
| 绿宝石 | 毒素伤害、毒素抗性、中毒时间缩短 |
| 骷髅石 | 物理伤害降低、魔法伤害降低 |
| 钻石 | 所有抗性、防御强化 |
| 紫宝石 | 力量、精力、敏捷 |

附魔几率与消耗：
- 附魔 1/2（第一次）：消耗 2～4 宝石点，成功率 5%
- 附魔 2/2（第二次）：消耗 3～6 宝石点，成功率 4%
- 更高次数的附魔：消耗更多，成功率更低

## 八、打孔系统 (Socket Punch)

### 8.1 通过镶嵌之球
**输入**：任意无孔武器/护甲 + 镶嵌之球 → 打 1 孔

### 8.2 通过拉苏克打孔公式
按装备品质和底材类型消耗不同材料打孔（1～6 孔，按底材最大孔数）。

**示例**：头盔(低质/普通/高质/魔法) + 魔法珠宝 → 1 孔
**稀有/暗金/套装**：同品质珠宝 + 固定孔数

## 十一、宝石袋系统 (Gem Bag)

**宝石袋** (`bag`) 是一个特殊的容器，用于存储所有类型的宝石点数。

**获取**：通过回收/打孔/制作等操作产生或消耗宝石点

**用途**：
