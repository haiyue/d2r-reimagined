# 职业指南：技能、属性与特长

> D2R Reimagined MOD 八大职业的属性成长、技能树与加点建议

---

## 属性系统概要

每升一级获得 **5 点属性**、**2 点技能**。

| 属性 | 作用 | 
|------|------|
| **力量 (Strength)** | 装备需求（盔甲、武器、盾牌） |
| **敏捷 (Dexterity)** | 装备需求、命中率、格挡率 |
| **体力 (Vitality)** | 生命值、耐力 |
| **能量 (Energy)** | 法力值 |
