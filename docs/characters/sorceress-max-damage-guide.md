## 二、为什么闪电系伤害最高？（模组机制分析）

### 2.1 模组特有的装备优势

D2R Reimagined 为闪电系提供了极为丰富的 **-敌人电抗** 堆叠手段：

| 来源 | 减电抗值 | 说明 |
|------|---------|------|
| **无限 Infinity**（佣兵武器） | -45~55% | 12级 Conviction 光环，对地狱怪物仍有 1/5 效果 |
| **新月 Crescent Moon**（副手武器） | -35% | 可在副手切换释放 Static Field 后切回 |
| **Haven of Light**（暗金法球） | -20~30% | 模组新增暗金，同时 +3 术士技能、+3 闪电精通 |
| **Naj's Ancient Vestige** 套装（4件） | -20% | 套装完全体提供全穿透 +5 闪电技能 |
| **Thought** 符文之语（法球 Tir+Lum+Ist） | -20% | 模组新增符文之语，+2 术士 |
| **电系破免板 Crack of the Heavens** | -70% | 地狱破免后生效（破免后抗性惩罚为原来的 1/5） |
| **Griffon's Eye**（权冠） | -20% | +15% 闪电技能伤害，减电抗 |

**总减电抗** 在破免后可达到 **-150%~-190%** 以上。这意味着即使地狱怪物有 117% 电抗（破免板将其降至 100% 以下后为 ~95%，再被 Conviction 降低后约 50~75%），闪电伤害仍能打出极高有效伤害。

### 2.2 闪电系技能优势

| 技能 | 优势 |
|------|------|
| **闪电 (Lightning)** | 穿透型法术，对直线目标打出全额伤害，无视格挡 |
| **连锁闪电 (Chain Lightning)** | 弹跳 20+ 次，清图之王 |
| **闪电精通 (Lightning Mastery)** | 满级 + 装备提供 **+500%~+700%** 闪电伤害乘数 |
| **无施法冷却** | 可持续施法，享受最高 200% FCR 档位（7帧/发，约 3.6 发/秒） |
| **静电力场 (Static Field)** | BOSS 削血神技，25% 范围内敌人血量减半 |

### 2.3 伤害计算公式

```
单发闪电伤害 = 基础技能伤害 × (1 + Lightning Mastery%)
有效伤害     = 单发伤害 × max(0, 1 - 怪物最终抗性)
最终 DPS     = 有效伤害 × 施法频率
```

**满级 (99级) + 毕业装备** 下：
- 闪电基础伤害（技能等级 ~45-48 级）：约 5,000~8,000
- Lightning Mastery 加成：~+650%~+750%
- 单发伤害：**~40,000~65,000**
- 施法频率（200% FCR 档位，7帧）：约 3.6 发/秒
- **理论 DPS：超过 200,000**

## 五、属性加点方案

| 属性 | 加点方案 | 说明 |
|------|---------|------|
| **力量 (Strength)** | **够装备** 约 100~120 点（裸体状态） | Spirit Monarch 需 156 力量，但装备上的 +力量可覆盖 |
| **敏捷 (Dexterity)** | **不加** | 术士不需要命中率和格挡（穿盾时可不加） |
| **体力 (Vitality)** | **剩余全加** | 每点体力 = 2 生命，满级约 1,200~1,500 裸体生命 |
| **能量 (Energy)** | **不加** | 靠装备 + Warmth + 佣兵 Insight 解决法力问题 |

### 属性点分配示例（99 级裸体）

```
初始属性：力量 10、敏捷 25、体力 35、能量 10
可分配属性：98 级 × 5 = 490 点

推荐分配：
  力量加到 + 装备加成后 ≥ 156（Spirit 需求）
  → 裸体力量约 80~100（约 70~90 点）
  敏捷：不加（0 点）
  体力：其余全加（约 380~400 点）
  能量：不加（0 点）
```

## 九、最终建议

### 推荐路线

```
首通 → 冰霜系（安全）
  ↓ 收集装备
火焰系（高效清图）
  ↓ 收集 Infinity + 极品装备
闪电系（伤害天花板）
```

### 流派选择决策树

```
你追求什么？
├─ 理论最高 DPS ──────→ ⚡ 闪电系（需要 Infinity + 破免板 + 高配）
├─ 最快成型开荒 ──────→ ❄️ 冰霜系（门槛最低，MF 首选）
├─ 最爽清图体验 ──────→ 🔥 火焰系（火球 + 陨石打击感极强）
└─ 8PP 地狱全能 ─────→ ⚡ 闪电系（群伤单伤兼顾）
```

### 一句话总结

> **D2R Reimagined 术士极限伤害：闪电系 > 火焰系 > 冰霜系。**  
> 模组提供了大量 -电抗装备（Crescent Moon -35%、Haven of Light -20~30%、Infinity -45~55% 等），  
> 配合 Lightning Mastery 的 +500~700% 乘数和 200% FCR 的 7 帧施法，  
> DPS 可轻松突破 20 万，地狱 8PP 也能横着走。
