### 2.2 什么是"破免"？为什么要破免？

#### 为什么需要破免

地狱难度中大量怪物拥有 **元素免疫**（抗性 ≥ 100%）：

| 怪物类型 | 常见免疫 | 地狱难度 |
|:--------|:--------|:--------|
| 沉沦魔法师、骷髅法师 | ⚡ 闪电免疫 | **电抗 120~140%** |
| 女伯爵、火焰强化怪 | 🔥 火焰免疫 | **火抗 110~150%** |
| 冰封地狱怪、冰骷髅 | ❄️ 冰霜免疫 | **冰抗 120~150%** |
| 魔法抵抗怪 | 🌀 魔法免疫 | **魔抗 100~120%** |

如果不破免，你的火焰/闪电/冰霜/魔法技能打在这些怪物身上 = **0 伤害**。

#### 破免的原理

"破免"就是通过特殊手段将怪物 100%+ 的抗性降至 **99% 以下**，使其不再免疫。

核心公式：

```
破免后抗性 = 原始抗性 - 破免效果
最终抗性 = 破免后抗性 - 降抗效果
```

**关键规则**：
1. **破免板（Sunder Charm）** 对免疫怪物 **全额生效**（-70% 抗性）
2. **审判光环（Conviction）** 和 **降低抵抗诅咒** 对免疫怪物 **仅 1/5 生效**
3. 免疫破除后（抗性 < 100%），审判和降抗恢复 **全额生效**
4. 装备上的 **-X% 敌人抗性** 始终全额生效（无论是否免疫）

#### 火焰术士实际破免计算示例

假设遇到一个 **火焰免疫** 的沉沦魔法师（火抗 120%），你有火焰裂隙 + 审判 5 级：

```
① 破免板生效（全额）：
   120% - 70% = 50%（不再免疫 ✓）

② 审判光环 Conviction 5 级（对原免疫怪 1/5）：
   -50% × 0.2 = -10%
   50% - 10% = 40%

③ 装备 -%敌人火抗（全额生效）：
   地狱炎 -10% + 地狱词典 -15% = -25%
   40% - 25% = 15%（最终抗性）

④ 有效伤害倍率：
   1 - 0.15 = 0.85（你的火焰伤害造成 85% 效果）
```

> 如果没有破免板：120% - 审判(50%×0.2=10%) - 装备25% = **85%** → 仍 ≥ 100%，**完全免疫 ❌**

#### 火焰术士如何获得降抗

| 方式 | 来源 | 对免疫怪效果 | 获取方式 |
|:----|:----|:----------:|:--------|
| **破免板** | 火焰裂隙（掉落/制作） | **-70%（全额）** ✅ | 地狱怪物掉落，或合成 |
| **审判光环** | 地狱词典 自带 +5 级 | -50%×0.2=**-10%** ✅ | 装备地狱词典即可 |
| **降低抵抗** | 深红法典 自带 +2~3 级 | -61%×0.2=**-12%** ✅ | 副手深红法典施放 |
| **装备减抗** | 地狱炎(-10%) + 地狱词典(-15~20%) | **全额生效** ✅ | 装备自带 |

**推荐破免流程**：
1. 刷地狱怪物掉落 **火焰裂隙**（或用地狱钥匙碎片合成）
2. 放在背包中 → 自动破免
3. 装备 **地狱词典** → 自动开启审判光环
4. 遇到火免精英 → 切副手深红法典 → 施放 **降低抵抗**
5. 合计降抗：-70%（破免）+ -10%（审判）+ -12%（降抗）+ -25%（装备）= **-117%**
6. 最终火抗从 120% 降至 **3%** → 末日打出 **全额伤害**

#### 瘴气术士需要破免吗？

瘴气术士主伤害类型为 **魔法伤害（Magic Damage）**。魔法免疫怪物相对较少，但确实存在（如部分魔法抵抗暗金怪）。

| 流派 | 需要破免板 | 推荐破免板 |
|:----|:---------:|:---------|
| 🔥 火焰术士 | **必须** | 火焰裂隙 |
| 🌀 瘴气术士 | 可选（魔法免疫少） | 漆黑裂痕 |
| 👹 召唤术士 | **必须**（佣兵带审判） | 根据主伤害类型选 |

> **一句话**：破免板是让免疫怪物变得可攻击的关键道具。**火焰术士必须带火焰裂隙**，否则遇到火免怪只能干瞪眼！
