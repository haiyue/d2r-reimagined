# Oskills & CTC Only Skills

## Oskills
### What is an Oskill?
OSkills (+X to Skill) only provides +3 to the class that owns the skill, total. This limit of +3 to owned skill includes multiple sources of the same OSkill. All other classes receive the full benefit.

If a +X to Skill also contains (Class Only) then only that class benefits from the skill level increase and gains the full effect.

More information can be found [Here](https://www.theamazonbasin.com/wiki/index.php/Oskill) and [Here](https://diablo.fandom.com/wiki/Oskill#Same_class_cap)

## CTC Skills

### What is ctc?
CTC stands for chance to cast. Which is just a mechanic that allows you to trigger another skill given a % chance and X skill level. 

Synergies do apply to CTC Skills. Skill levels of the CTC skill cannot be increased, however you can stack the % chance by equipping multiple items with the same CTC skill and event type.

There are multiple ways(events) to trigger a CTC skill: On attack, on striking(landing a successful hit), when struck, when you die, when you level up, and when you kill an enemy.

Note1: CTC when you kill an enemy only works with weapon damage, weapon or poison damage, or open wounds. Spells will not proc CTC skills.
Note2: When struck means when you actually take a hit. Blocking does not proc when struck CTC. 

For more detailed information see the amazon basin page: [Here](https://www.theamazonbasin.com/wiki/index.php/Chance_to_Cast)
