Use a Launcher or Manually Install Reimagined

 

Reimagined Launcher (Very Easy) Installs in Windows & automatically updates itself, available on Nexus Mods.  With one-click it can find the latest version of Reimagined online too. Auto backup for your saved games.

 

Beta Launcher (Easy): Is a batch file to run, shown in pictures below in this guide.  Auto backup for your saved games.  

 

Manually follow-the-steps (Easy/Hard): You will lookup your own installation directory & you'll be responsible for backing up your own saved games incase a crash occurs during a loading screen.

 

Default File Locations - Depends if you bought on Battle.net App or Steam

C: Drive is the most common install location. Yours may differ depending on your setup.  Launcher finds them automatically.

Battle.net D2R Mod Files: C:\Program Files (x86)\Diablo II Resurrected\mods\Reimagined

Steam D2R Mod Files: C:\Program Files (x86)\Steam\steamapps\common\Diablo II Resurrected\mods\Reimagined

Saved Game Files: C:\Users\%USERPROFILE%\Saved Games\Diablo II Resurrected\mods\ReimaginedThree

 

Manually Create Saved Game Backups

1. In File Explorer, navigate to your Saved Game Files directory as listed above.

2. Go back one directory to the \Saved Games\Diablo II Resurrected\mods\ folder which contains the saved games for each mod as shown in the picture below.

All Reimagined 3.x saved game versions will be in the ReimaginedThree folder.

3. As shown in the picture right click on the  ReimaginedThree folder, select ‘Send to’ and choose Compressed (zipped) folder which copies the contents into a ZIP file. Rename the ZIP file to something memorable, and move it somewhere easy to remember (e.g. your desktop).

Remember to periodically create new ZIP saved game backups so that you don't lose progress in case of crashes.  While playing the game, if your PC crashes while the game is writing to a saved game that game can become corrupt resulting in a “Failed to join game” message.  The Reimagined Launcher is capable of backing up your saved games making this easier.

Your Saved Game files are unaffected by these steps.

2. Move the downloaded .zip file from your Downloads folder into the Reimagined BETA Launcher folder (wherever you saved it, such as on the desktop).

3. Run the launcher.

  • If you see a warning from Windows Defender, press ‘More Info’ and click ‘Run Anyway’.

6.  Once the Reimagined Mod Install is completed you can press any key to restart the launcher. 

 

7.  You can now type 1 to pick 'Play Reimagined' or you could create a shortcut link to the game following the steps below.

  • Windows11 Users Click Show More Options to complete the above step.

Crashing Frequently? Tips to Troubleshoot

Here are some common resolutions to frequent crashing we've encountered in bug reports across the Discord, Nexus and Reddit.

1. Reimagined 3.0+ is not compatible with saves from Reimagined 1.x.x, Reimagined 2.x.x, D2R vanilla, or other mods if the save file contains items. Trying to place them in the saves folder and use them will often cause the game to crash.

2. Classic characters cannot be created and are incompatible with the mod.

3. Change graphics option VFX Lighting Quality off from Ultra to High or Medium as it barely has any visual impact but kills game performance and causes crashing.

4. Cap your framerate to your monitors refresh rate and/or use Vsync.

5. Don't Alt + Tab, (click out of the screen if playing in windowed) the game doesn't enjoy being out of focus. You can try to mitigate this crash by limiting the Background FPS of D2R.exe via your GPU Control Panel. (The game still likes to crash anyways but the background FPS limit helps reduce the frequency.)

6. Don't equip items on Mercs with Chance to Cast a minion. (Reanimate is fine, it is specifically CTC that seems to cause issues.)

7. Change Power Settings from Performance to Balanced (Unintuitive but seems to work for some people.)

8. If you are using other mods, you can try disabling them to see if the crashes continue but if they edit itemstatcost.txt then disabling them will cause crashes anyways so pay attention to what you are doing and make backups before experimenting.

9. If you are making local mod edits, undo those edits or reinstall the mod fresh.

 

D2RMM

Using a Mod Manager allows you to easily install a variety of mods together (assuming they play nice) without having to know anything about modding.

Reimagined is packed full of expansive features and QOL updates, so installing extra mods isn't always necessary. 

Check the list of Reimagined features below to see if a 3rd-party mod is still necessary.

 

When using D2RMM your active mod files will be now be located under the D2RMM\mods\ folder instead of Diablo II Resurrected\mods\.

Any changes you make to Reimagined files needs to be done in the D2RMM mods directory.

If using the BETA Launcher (above), you will need to copy the file changes over from Diablo II Resurrected\mods\Reimagined\Reimagined.mpq\ to D2RMM\mods\Reimagined\.
 

File Steps

1. Download D2RMM from D2RMM Github.

2. Open the D2RMM folder - navigate to the mods directory.

  • You should see a text file called 'PUT YOUR MODS HERE' - leave this window open, we'll use it later.

3. Open a new File Explorer window and navigate to your Reimagined mod folder under your D2R Install directory.

4. Copy the Reimagined.mpq folder (this should contain a Data folder and ModInfo.json).

5. Paste the Reimagined.mpq folder into the D2RMM/mods folder from step 2.

  • We recommend renaming the folder to Reimagined

D2RMM Steps

6. Open D2RMM as Administrator.

7. Once open, click the three dots in the bottom-right corner and select 'Refresh Mod List'.

8. Reimagined should appear in the mod list - click the box on the left-side to enable it.

The Reimagined folder must always be first in the list of mods. Click and drag the line to the top if it isn't.

9. Navigate to the Settings tab (at the top).

  • 9a. Click on General.
    • Change 'Output Mod Name' to 'Reimagined'.
    • Toggle 'Automatically install mods before running the game'.
  • 9b. Click on Game Launcher.
    • Under 'Extra Game Args', paste the following: -mod Reimagined -txt
    • Feel free to add any other args you want after everything else.

10. Navigate back to the Mods tab.

11. Click Install Mods.
 

Play time!

12. Hit 'Run D2R'.
 

Loot Filters

This guide assumes you have already downloaded and installed D2RMM (see above)

  • Because of the Warlock DLC we do recommend just using the in-game loot filtering system that is compatible with our treasure file, strings, and mod-added items.
  • For users without the DLC, the filter below and others like it on Nexus are use at your own risk and not supported by Reimagined. The mod changes every file each of the filters touches and authors would have to make a specific version for use with Reimagined for it to be fully compatible with no issues.
     

File Steps

1. Download Pseudo Loot Filter from Nexus Mods.

  • Select ‘Mod manager download’ to automatically download and load the mod into D2RMM.
  • Select ‘Manual download’ to download the mod to your Downloads folder - extract the mod into your D2RMM/mods folder.
     

D2RMM Steps

2. In D2RMM, click the 3 dots in the bottom-right, and select ‘Refresh Mod List’.

3. You should now see Pseudo Loot Filter appear in the list of mods - click the box to the left to enable the mod.

4. To configure the loot filter, click ‘settings’ to the right.

5. All items are enabled by default (the slider is on the right-side) - clicking the slider DISABLES the item (this affects all rarities, white, magic, rare, set, unique).

  • Disabling an item simply means that when a disabled item would drop, the game chooses to drop nothing instead.
  • Disabling items does not affect drop chances of other items.
  • It may take a few runs to tweak this list to your liking.

6. Once you've finished disabling undesirable items, click out of the menu.

7. Click ‘Install Mods’ in the bottom-right.

8. Hit ‘Run D2R’.

Reimagined and D2RLoader

D2RLoader provides LAN TCP/IP support for Reimagined, including Reign of the Warlock (ROTW), without the former class and mechanic restrictions.

notice: D2RLoader is still in active development, so installation steps and configuration options may change in future updates.

Before you begin: Everyone joining the same TCP/IP game must use matching D2R, D2RLoader, and Reimagined compatibility-build versions.

1. Install the Reimagined compatibility build

Download the Reimagined compatibility build from Google Drive.

Important: The official Reimagined release on Nexus is not directly compatible with D2RLoader. Use the compatibility build linked above.

Download and extract the archive, then place the extracted Reimagined folder in:

Diablo II Resurrected\mods

2. Install and configure D2RLoader

Open the D2RLoader link, then download the latest release from the project's Download category.

After extracting the archive, move the d2rloader folder, D2RCore.dll, and D2RLoader.exe into the Diablo II Resurrected installation folder containing D2R.exe.

Open d2rloader\config\d2rloader.toml in a text editor and set Reimagined as the default mod:

default_mod = "Reimagined"

Save the file with Ctrl+S, close the editor, and launch the game by running D2RLoader.exe.

3. Set up Radmin VPN for LAN play

The game and loader are now ready. To play TCP/IP with friends over the internet, every player must join the same virtual LAN.

Download and install Radmin VPN, then turn it on.

Open the Network menu. Create a private network for your group, or select Join Network to connect to an existing one.

Example public network

Network name: D2RR TCP/IPPassword:     DeckardLikesWine@69

Enter the details and click Join. Radmin VPN should display the shared network and its connected players.

4. Connect in-game through TCP/IP

From the character-selection screen, click TCP/IP at the bottom center.

  • Hosting: Click Host Game. Give the other players your Radmin VPN IP address.
  • Joining: Click Join Game, enter the host's Radmin VPN IP address, and click OK.

Multiplayer checklist

  • The same D2R version
  • The same D2RLoader version and configuration
  • The same Reimagined compatibility-build version
  • Matching mod files

Mismatched versions or files may cause connection failures, crashes, missing visuals, broken skills, or other unexpected behavior.
